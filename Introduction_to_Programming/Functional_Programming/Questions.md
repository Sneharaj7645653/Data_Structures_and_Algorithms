# Basic Math and Function Programming Problems

## Q1. Factorial of N - Functions

### Problem Description
Given a number `N`, return factorial of `N`.

Formula:
```
n! = n * (n-1) * (n-2) * ... * 3 * 2 * 1
```
Example: `5! = 5 * 4 * 3 * 2 * 1 = 120`

Corner Case:
```
0! = 1
```

### Problem Constraints
```
0 <= N <= 10
```

### Input Format
Argument `A` which is given number `N`.

### Output Format
Return the answer from factorial function.

### Example Input
```
Input 1:
5

Input 2:
6
```

### Example Output
```
Output 1:
120

Output 2:
720
```

## Q2. Add the numbers

### Problem Description
Given two numbers `A` & `B`. Return their sum.

### Problem Constraints
```
1 <= A <= 10^7
1 <= B <= 10^7
```

### Input Format
First argument is an integer `A`.  
Second argument is an integer `B`.

### Output Format
Return an integer which is the sum of `A` & `B`.

### Example Input
```
A = 4, B = 3
A = 14, B = 34
```

### Example Output
```
7
48
```
